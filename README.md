# Document Editor

A production-ready rich text document editor built on canvas rendering technology.

## Features

- Full rich-text editing with canvas-based rendering
- Multiple paper sizes and orientations
- Headers, footers, and page numbering
- Tables, images, hyperlinks, and LaTeX formulas
- Watermarks, code blocks, and content controls
- Search and replace with regex support
- Multiple editor modes (Edit, Read Only, Form, Print, Design, Graffiti)
- Print and PDF export
- Signature support
- Fully internationalised in English

## Getting Started

### Prerequisites

- Node.js >= 16.9.1

### Install

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```
